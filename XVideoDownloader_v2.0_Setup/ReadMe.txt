============================================
X VIDEO DOWNLOADER v2.0 - USER GUIDE
============================================

FIRST TIME SETUP:
1. Double-click "XVideoDownloader.exe"
2. Follow the Setup Wizard instructions:
   - System requirements will be checked
   - Playwright browsers will be installed (~300 MB)
   - You'll configure your database settings
3. After setup, the application will launch automatically

DAILY USE:
1. Double-click "XVideoDownloader.exe"
2. Click "Re-Login" if needed (first time or if cookies expired)
3. Select download mode (Latest Posts or Last N Days)
4. Set your parameters
5. Click "Run Script"
6. Watch progress in real-time!

FEATURES:
✓ Download videos from X/Twitter bookmarks
✓ Multi-threaded downloads (faster!)
✓ Automatic unbookmarking after download
✓ HTML reports for tracking
✓ Real-time progress display
✓ Resume capability

REQUIREMENTS:
- Windows 10/11
- Internet connection
- Google Chrome browser (recommended)
- MySQL database access

TROUBLESHOOTING:
- If setup fails, close all Chrome windows and try again
- Check firewall isn't blocking the application
- Verify database credentials in .env file
- Check logs/ folder for error details

SUPPORT:
For issues, check the error log in:
logs/errors_[timestamp].log
