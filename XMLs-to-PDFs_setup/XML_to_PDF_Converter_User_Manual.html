  • Each PDF has the same name as its source XML file

Troubleshooting

Common Issues & Solutions

‚ùå Problem: "No XML files found" message

Solution: Ensure the folder you selected contains files with .xml extension (case-insensitive).

‚ùå Problem: Empty or incomplete PDFs

Solution: This typically means the XML file had no text content. The program will note "(No text content found)" in such cases.

‚ùå Problem: Conversion errors for specific files

Solution:

  • Check that the XML file is valid (not corrupted)
  • Ensure the file is not open in another program
  • Check file permissions

‚ùå Problem: Program won't start

Solution:

  • Ensure you have administrator rights
  • Check Windows Defender or antivirus hasn't quarantined it
  • Try running as administrator (right-click ‚Üí Run as administrator)

‚ùå Problem: Slow conversion

Solution:

  • Close other programs to free up system resources
  • The program automatically uses optimal threading
  • Very large XML files naturally take longer

Tips & Best Practices

üí° Tips for Best Results

  1. Organize Your Files: Keep XML files you want to convert in a dedicated folder
  2. File Names: Use descriptive XML file names as they become the PDF file names
  3. Batch Processing: Convert multiple files at once for efficiency
