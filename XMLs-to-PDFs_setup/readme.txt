                    XML to PDF Converter v1.0
           Batch Convert XML Files to Formatted PDFs
================================================================================

QUICK START:
------------
1. Run XML_to_PDF_Converter.exe
2. Click "Select XML Folder" and choose your folder
3. Click "Convert All"
4. Find your PDFs in the "PDFs" folder

WHAT IT DOES:
-------------
This program converts XML files into readable, formatted PDF documents.
It processes multiple files simultaneously for maximum speed.

SYSTEM REQUIREMENTS:
--------------------
- Windows 7 or later
- No additional software needed

WHERE ARE MY PDFS?
------------------
PDFs are saved in a folder called "PDFs" in the same location as this program.

Example:
  Program location: C:\Users\YourName\Desktop\XML_to_PDF_Converter.exe
  PDFs saved to:    C:\Users\YourName\Desktop\PDFs\

FEATURES:
---------
✓ Batch conversion - process hundreds of files at once
✓ Parallel processing - uses all your CPU cores for speed
✓ Deep scanning - captures all XML content
✓ Formatted output - color-coded, easy-to-read PDFs
✓ Progress tracking - see conversion status in real-time
✓ Error reporting - know which files had issues

TIPS:
-----
• Keep XML files in one folder for easy batch conversion
• Original XML files are never modified
• The program works offline - no internet required
• Close the program completely before moving it to a new location

TROUBLESHOOTING:
----------------
Problem: "No XML files found"
Solution: Make sure your folder contains .xml files

Problem: Program won't start
Solution: Right-click and "Run as administrator"

Problem: Windows Defender warning
Solution: This is a new program; click "More info" then "Run anyway"
