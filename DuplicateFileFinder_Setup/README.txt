Duplicate File Finder
=====================

Version: 1.0.0

Description:
-----------
A powerful tool for finding and managing duplicate images and videos on your computer.

Features:
--------
• Exact duplicate detection using file hashing
• Visual similarity detection (70-90% threshold)
• Filename pattern matching
• Safe quarantine system (no permanent deletion)
• Interactive HTML reports with thumbnails
• Support for images: JPG, PNG, GIF, BMP, WebP, TIFF, ICO
• Support for videos: MP4, AVI, MOV, MKV, WMV, FLV, WebM, M4V, MPG, MPEG

How to Use:
----------
1. Launch the application
2. Click "Browse" to select a folder to scan
3. Click "Scan for Duplicates"
4. Review results in your web browser
5. Select duplicates to delete (they go to quarantine)
6. Use the Reset button when done

Quarantine Location:
------------------
Deleted files are moved to:
%USERPROFILE%\DuplicateFiles_Quarantine

Each deletion session creates a timestamped folder with a deletion log.

System Requirements:
------------------
• Windows 7 or later
• 4GB RAM minimum (8GB recommended for large scans)
• Python is NOT required (bundled)
